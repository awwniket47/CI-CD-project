How it works

One query. Three agents.
A complete report.

The system handles everything automatically — from searching the web to writing and storing the final report.

STEP 01
💬
You ask a question
Type any retail industry question — trends, technologies, company strategies, market analysis. The system understands natural language.
STEP 02
🔍
Agents search and analyse
The Researcher searches Tavily for articles. The Analyst reads everything and extracts the 5–7 most important insights with statistics and real company examples.
STEP 03
📄
Report saved to knowledge base
The Writer produces a 600+ word professional report. Saved to ChromaDB as a vector embedding and as a .txt file — semantically searchable forever.
Features

Everything you need
for retail research

Built for depth, speed, and knowledge accumulation.

⚡
Live Agent Pipeline
Watch each agent work in real time via Server-Sent Events. See log lines as they happen — no page refresh needed.
🧠
Gemini-Powered Analysis
Google Gemini 2.0 Flash reads search results, extracts structured insights, and writes professional reports with citations.
🔍
Semantic Search
ChromaDB stores every report as a vector embedding. Find conceptually related reports even without exact keywords.
💾
Dual Knowledge Store
Every report is saved twice — in ChromaDB for semantic search, and as a .txt file for human-readable backup.
📋
Structured Reports
Every report follows a consistent 7-section format — Executive Summary, Key Findings, Trends, Examples, Challenges, Outlook, Sources.
📱
Research History
All past sessions are saved. Revisit any previous research, read the full report, and review the agent logs.
